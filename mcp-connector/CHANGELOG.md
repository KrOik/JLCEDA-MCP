# Changelog

本文件记录 MCP 连接器的版本变更。

## [1.0.0] - 2026-03-16

### 当前功能

**扩展管理**

- 扩展安装后随嘉立创 EDA 启动自动激活，无需手动触发。
- 在首页、原理图、PCB 顶部菜单栏注入「MCP 连接器」菜单，提供「连接设置」和「About」入口。

**连接设置页面**

- 提供独立的连接设置对话框（500×310，可最小化收起）。
- 支持配置桥接 WebSocket 地址（`ws://` 或 `wss://` 格式），默认地址为 `ws://127.0.0.1:8765/bridge/ws`。
- 地址格式实时校验，非法格式拒绝保存并提示错误。
- 设置页实时展示两个状态胶囊：桥接服务连接状态和 WebSocket 连接状态。
- 配置变更后自动通知桥接运行时重新连接。

**桥接 WebSocket 客户端**

- 扩展激活后自动建立到服务端的 WebSocket 长连接，连接失败后每 1.2 秒自动重连。
- 连接建立超时（8 秒）和应用层握手超时（5 秒）双重保护，超时后触发重连。
- 心跳保活：每 1 秒发送心跳，5 秒内未收到服务端任何消息则判定连接超时并重连。
- 支持 `active`（活动）/`standby`（待命）角色裁决，仅活动角色执行桥接任务，待命角色等待。
- 每次建立新连接使用唯一 socketId，避免与残留连接冲突。
- 当前页面不是原理图或 PCB 编辑页时，连接自动暂停。

**桥接任务处理**

- `jlceda_api_search`：读取扩展内置离线 API 文档，按关键词执行检索；支持关键词拆分、倒排索引命中与评分降级回退；支持 `scope`（`callable` / `type` / `all`）和 `owner` 名称过滤；单次最多返回 50 条结果。
- `jlceda_api_invoke`：将 AI 下发的 `apiFullName` 和参数解析为实际 EDA API 调用；支持 `positionalArgs`、`args`、`namedArgs`（含 `parameterOrder`）三种参数格式；防 prototype pollution 拦截；结果异步序列化后返回。
- `jlceda_context_get`：采集当前 EDA 运行时快照，包含当前工程、文档、面板、原理图、原理图图页、PCB、Panel 信息，以及原理图和 PCB 已选中图元 ID 列表。

**上下文同步**

- 每 1 秒周期性将上下文快照同步到桥接服务端，确保服务端持有最新 EDA 状态。

**日志系统**

- 统一日志结构，包含时间、级别、来源模块、事件、摘要、消息、桥接地址、客户端 ID、租约、错误码等字段。
- 客户端日志上报队列，最多缓冲 200 条；日志上报受调试开关控制，关闭后停止发送。
- 支持日志去重（1.5 秒窗口）和噪音日志节流（8 秒窗口），防止高频事件刷屏服务端。

**调试开关**

- 服务端可动态下发调试开关，控制连接信息日志和系统日志的采集与上报。
